## Album List
Cправочник по музыкальным альбомам

Технологии:
1. Laravel, PHP
2. Bootstrap
3. PostgreSQL (на хосте SQLite)
4. Deezer API - https://developers.deezer.com/api

Страницы:
1. home - http://artymsamoh.temp.swtest.ru/. Список всех пластинок. Удаление доступно только авторизированному пользователю
2. edit - http://artymsamoh.temp.swtest.ru/edit. Доступ только авторизированному пользователю. После отправки формы проверка существования альбома по названию и исполнителю в Deezer API
3. login - http://artymsamoh.temp.swtest.ru/login.
4. register - http://artymsamoh.temp.swtest.ru/sign-up.
